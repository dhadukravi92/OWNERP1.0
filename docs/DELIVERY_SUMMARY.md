## 🎨 DESIGN SYSTEM AT A GLANCE

### Color Palette (Semantic)
```
🔵 Primary Blue: #3d7fff      (Actions, primary CTAs)
🌊 Teal: #06b6d4              (Secondary actions, info)
🟢 Success: #22c55e            (Gains, approvals)
🔴 Danger: #ef4444             (Losses, errors)
🟠 Warning: #f59e0b            (Pending, caution)
⚫ Dark BG: #0a0d14             (Dark mode)
```

### Typography (Fibonacci)
```
55px  ← Page titles
34px  ← Section headings
21px  ← Subsections
13px  ← Body text (base)
11px  ← Captions
```

### Spacing Grid (4px base)
```
4px   8px   12px  20px  32px  52px  84px
1×    2×    3×    5×    8×    13×   21× (Fibonacci!)
```

### Button Sizing
```
xs:  6×12px padding, 24px height
sm:  8×12px padding, 32px height
md:  12×16px padding, 40px height (default)
lg:  16×24px padding, 48px height
xl:  20×32px padding, 56px height
```

## 💡 THE MATHEMATICAL DIFFERENCE

This design system applies:
- **Golden Ratio (φ=1.618)** for card proportions and layouts
- **Fibonacci Sequences (1,2,3,5,8,13,21,34,55)** for spacing and typography
- **Hick's Law** to reduce cognitive load (decision time = log₂(choices))
- **Semantic Color Encoding** (green=gain, red=loss, amber=caution)

**Result:** A design that feels intuitively "right" without users knowing why.
